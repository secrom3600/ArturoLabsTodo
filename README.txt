pasos para iniciar

1. Ejecutar Docker Desktop

2. iniciarlizar repositorios por uno de los dos siguientes pasos:

    2.A. Ejectuar "docker compose up --build -d" (en una consola) dentro de la carpeta ArturoLabs---Backend y 
        luego en la carpeta ArturoLabs---Frontend
    2.B. Ejectuar el archivo "Iniciar_Windows.bat" o "Iniciar_Linux.sh", depende del Sistema Operativo que se use

Atencion: Si ya ah ejecutado alguna version con anterioridad o quiere reiniciar al estado base del sistema, 
        tendra que ejecutar "docker compose down" en ambas carpetas o utilizar el archivo "Limpiar_windows.bat" 
        o "Limpiar_Linux.sh" 

3. La pagina se encuentra en la URL "http://localhost:8050"
